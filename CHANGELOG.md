# 更新日志

## 0.1.14

- 关闭启动时自动检查更新，避免软件一打开就触发下载。
- 检查更新只负责提示新版本，不再自动下载或自动安装。
- 更新区域将“重启安装”改为“下载新版”，点击后打开 GitHub 最新发布页手动下载安装。
- 避免未签名 macOS 包自动安装失败导致误显示“版本号获取失败”。

## 0.1.13

- 修复 macOS 检测更新时提示“版本号获取失败”的问题。
- macOS 发布包新增 zip 更新包，供 Electron 自动更新使用。
- DMG 继续保留给用户手动下载安装。

## 0.1.12

- 更新区域新增“更新日志”按钮，可在软件内查看最新版本说明。
- 更新日志弹窗会展示当前版本、最新版本、发布时间和 GitHub Release 正文。
- 弹窗支持刷新日志和打开 GitHub 发布页。
- 发布流程会优先读取本文件中对应版本的更新内容，方便后续版本写清楚改了什么。
